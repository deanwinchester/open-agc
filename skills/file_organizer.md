# 文件整理技能
当用户需要整理文件或清理目录时：
1. 使用 `execute_shell` 运行 `ls -la` 或 `find` 查看目标目录结构
2. 分析文件类型分布：
   ```bash
   find /path -type f | sed 's/.*\.//' | sort | uniq -c | sort -rn
   ```
3. 根据用户需求提供整理方案：
   - **按类型分类**：文档(.pdf/.docx) → docs/，图片(.jpg/.png) → images/
   - **按日期分类**：按创建/修改时间分入 2024-01/, 2024-02/ 等
   - **按项目分类**：识别项目文件夹结构，合并分散的相关文件
   - **清理临时文件**：清理 .DS_Store, __pycache__, .log 等
4. 先展示整理计划，获得用户确认后再执行
5. 使用 `execute_shell` 执行 `mkdir -p` 和 `mv` 命令

⚠️ 重要：移动文件前务必确认，避免误操作。建议先做一个dry-run预览。
