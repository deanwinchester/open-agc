🐼 设置

🔑 API 密钥

配置各厂商的 API Key，未填写的厂商将不可用。

🤖 默认模型

选择主力模型，以及在主力不可用时的备选回退顺序。

当默认模型不可用时，将按此顺序依次尝试。

✨ 后台巡视 (Heartbeat)

开启后，智能体会定期在后台自动检查系统状态，一切正常则静默不打扰。

📁 沙箱安全 (Sandbox)

将智能体的文件访问和执行权限限制在特定目录内。

智能体将只能在该目录下读取/写入文件以及执行命令行指令。

📧 邮件监听与助手 (Email Listener)

配置智能体邮箱，允许它接收主人指令并代发邮件、总结邮件等。

🛠️ 技能管理

控制 Agent 可调用的技能开关，关闭的技能不会被加载。

🎭 多智能体配置 (Multi-Agent Profiles)

配置不同的智能体角色，可以分配不同的系统提示词。

🔌 Model Context Protocol (MCP)

配置外部 MCP Server。请填写标准的 JSON 配置 (兼容 Claude Desktop)。重启应用后生效。

⚠️ 需要系统权限

该操作需要额外的系统权限。请在终端中执行以下命令授权：

chmod +x ./start.sh
